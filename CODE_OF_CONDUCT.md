# Código de conducta

Este espacio busca aprendizaje colaborativo y respeto.

## En resumen
- Respeto en todo momento.
- Críticas: siempre sobre ideas y código, no sobre personas.
- Si algo incomoda o hay conflicto: reportar a moderación.

## Moderación
La moderación puede eliminar contenido o bloquear cuentas ante conductas repetidas.
