# Cómo contribuir

## Flujo recomendado
1) Si es una duda o aprendizaje → Discussion en `knowledge-base`.
2) Si es una tarea concreta → Issue (con pasos claros).
3) Si es una mejora reutilizable → PR a `snippets` o al repo correspondiente.

## Convenciones
- Títulos claros.
- Adjunta pasos para reproducir.
- Incluye capturas/logs cuando aplique.

Gracias por aportar.
